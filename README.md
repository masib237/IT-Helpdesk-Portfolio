# Windows Active Directory Home Lab

## Overview
This project is a hands-on Active Directory home lab built using VirtualBox to simulate a small enterprise IT environment. It covers the setup and administration of Windows Server infrastructure, including domain services, user and group management, Group Policy configuration, file sharing, and basic helpdesk troubleshooting scenarios.

The lab was designed to replicate common tasks performed in a Tier 1 / Junior IT Support role.


## Lab Progression

- LAB 0: Built and configured virtual environment using VirtualBox (Windows Server + Windows 11 client)
- LAB 1: Installed and configured Windows Server operating system
- LAB 2: Promoted server to Domain Controller and configured Active Directory Domain Services (AD DS)
- LAB 3: Created and managed Organizational Units (HR, IT, Sales), users, and security groups
- LAB 4: Joined Windows 11 client machine to the domain and verified authentication
- LAB 5: Performed user administration tasks including password resets, account unlocks, and user property management
- LAB 6: Configured Group Policy Objects (GPOs) to apply domain-wide and user-specific policies
- LAB 7: Implemented file sharing with NTFS and share-level permissions for access control
- LAB 8: Simulated IT helpdesk tickets including login issues, access problems, and system troubleshooting
- LAB 9: Documented the full lab environment

## Skills Demonstrated
- Windows Server installation and administration
- Active Directory Domain Services (AD DS) configuration
- Domain controller setup and management
- User lifecycle management (creation, modification, password resets)
- Security group and OU structuring
- Group Policy configuration and enforcement
- File sharing and NTFS permission management
- Basic network troubleshooting (DNS, domain join issues)
- IT support ticket simulation and incident resolution
- Technical documentation and lab reporting

## Purpose
The goal of this project was to build practical, job-relevant experience in IT support and system administration by simulating real enterprise scenarios in a controlled lab environment.

Each lab folder contains step-by-step documentation and screenshots showing configuration processes, troubleshooting actions, and outcomes.
